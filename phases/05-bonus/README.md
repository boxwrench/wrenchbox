# Phase 5: Bonus System (Milestone)

Snapshot of Phase 5 completion - combo detection with animated rewards.

## Features
- Detect specific sound combinations
- Animated overlay when bonus triggers
- 5 bonuses with unique animations
- Bonus indicator badges in header
- Repeatable vs one-time bonuses

## Bonuses
| Name | Combo | Animation |
|------|-------|-----------|
| Full Beat | kick + snare + hihat | pulse |
| Melody Master | bass + lead | wave |
| Low End Theory | kick + bass | shake |
| Groove Machine | kick + snare + bass | disco |
| Full Band | all 5 sounds | fireworks |

## Tech Stack
- Vanilla HTML/CSS/JS
- Tone.js v14.8.49
- CSS keyframe animations
- Particle system for effects

## To Run
Open `index.html`, drag sounds to slots, watch for bonus overlays.
