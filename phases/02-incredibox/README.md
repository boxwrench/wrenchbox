# Phase 2: Incredibox Core (Milestone)

Snapshot of Phase 2 completion - drag-drop interface with mute/solo controls.

## Features
- 7 empty character slots (drag to assign sounds)
- Drag sound icons from palette to slots
- Drag filled slots to palette to remove
- Drag between slots to move/swap
- Mute (M) button - silences slot, pattern continues
- Solo (S) button - mutes all other slots (exclusive)
- Quantized activation - sounds start on next bar boundary
- 5 synthesized sounds via Tone.js
- Keyboard shortcuts (1-7 mute, Space/Esc reset)

## Tech Stack
- Vanilla HTML/CSS/JS
- Tone.js v14.8.49 (CDN)
- HTML5 Drag and Drop API

## To Run
Open `index.html` in a browser, click to start, drag sounds to slots.
